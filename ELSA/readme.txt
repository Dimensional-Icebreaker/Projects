To run the program in your terminal, you can just stay in this directory and the command is  "ant build" to compile into the runnable and "ant run" to launch the program
New store will be automatically created each time you open the program.You can open a new store anytime once you are in the program.
You can create and customize your customer,order,computer and it's accesories in your own way
You can save all the data if you want to and open the existing store anytime you want
You can own your virtual storefront!
